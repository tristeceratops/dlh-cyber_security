# Explanations

Short descriptions of the small shell exercises in this folder.

- `0-bring_me_home`: uses `cd ~` to change to the home directory.
- `1-listfiles`: `ls -l` — long listing of files.
- `2-listmorefiles`: `ls -la` — long listing including hidden files.
- `3-back`: `cd -` — change back to the previous directory.
- `4-lists`: `ls -la . .. /boot` — list current, parent and `/boot` directories.
- `5-copy_html`: `cp -u *.html ..` — copy updated HTML files to parent (`-u` updates only when source is newer).

These exercises show basic shell navigation and common `ls` / `cp` flags used in everyday tasks.
